# Archive — completed & dropped projects

Frozen project files worth keeping for their decision history. Newest first.

When you archive a project (see the lifecycle table in [../README.md](../README.md)):
1. Move the project file here.
2. Freeze it — add a `Completed:` (or `Dropped:`) date and set the final status.
3. Remove its section from [../STATUS.md](../STATUS.md).
4. Add a one-line row below.
